## Library

> **Note:** At the moment it's in pre-alpha state.

A Continuous Time Bayesian Networks Library written in Rust. 🦀

## Develop

**Prerequisites:**

+ `rust`

_Prepare_ the development environment:

```sh
cargo build
```

That's all! ☕️

## Testing & Linting

To launch **tests**:

```sh
cargo test
```

To **lint**:

```sh
cargo check
```
