### From Source

**Prerequisites:**

+ `rust`

**Install:**

At the _root_ of the project's folder launch:

```sh
cargo install --locked --path .
```

**Uninstall:**

Anywhere in the system launch:

```sh
cargo uninstall rustyCTBN
```

**Upgrade:**

At the _root_ of the project's folder launch:

```sh
cargo install --force --locked --path .
```

## Develop
